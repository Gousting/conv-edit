# 三角洲行动 — 游戏理解与剪辑指南

> 本文档供 conv-edit 在剪辑三角洲行动素材时参考。
> 目标：让 AI 理解这个游戏的玩法、术语、内容生态，
> 从而生成符合玩家审美的解说字幕和剪辑节奏。

---

## 一、游戏基础

| 项目 | 内容 |
|------|------|
| 全称 | 三角洲行动 (Delta Force) |
| 引擎 | Unreal Engine 5 |
| 开发商 | 腾讯天美J3 琳琅天上 |
| 上线时间 | 2024年9月26日（国服） |
| 平台 | PC / 移动端 / 主机 |
| 当前赛季 | S10（2026年7月，前序 S8「蝶变时刻」、S9「回声」） |

### 三大模式

| 模式 | 英文 | 核心玩法 | 人数 |
|------|------|---------|------|
| **烽火地带** | Hazard Operations | 搜打撤：进图搜物资→接敌→撤离 | 小队（1-3人） |
| **全面战场** | Havoc Warfare | 海陆空载具大战场 | 24v24 |

## 二、核心术语

### 游戏内

| 术语 | 含义 | 使用场景 |
|------|------|---------|
| 撤离 | extract | 活着带物资出去。没撤离=白打。 |
| 摸金 | looting | 搜刮物资，高价值物品叫"肥肥" |
| 肥肥/大金 | high-value loot | 价值百万柯恩币的物品 |
| 柯恩币 | in-game currency | 游戏货币，简称"币" |
| 保险箱 | secure container | 死了也能保留其中物品 |
| 干员 | operator | 可选用角色，各有技能 |
| 配装 | loadout | 进图前选装备的界面 |
| 航天基地 | Space Base | 经典地图，也是破产兄弟视频的地图 |
| 核电站 | Nuclear Plant | S10新地图 |
| 绝密 | classified | 高难度高回报图 |

### 社区/直播用语

| 术语 | 含义 |
|------|------|
| 破产 | 连续死亡亏钱，柯恩币归零 |
| 自救/回本 | 靠队友或自己打回来 |
| 得吃 | 成功撤离+高额收益（来自广西话） |
| 白给 | 进图就死，什么都没带出来 |
| 夺舍 | 击杀敌人后拿走其装备 |
| 蹲/蹲坑 | 在一个位置长时间等待 |
| 对枪 | 正面枪战 |
| 拉枪线 | 战术移动，制造射击角度 |
| 被偷/被阴 | 被看不见的敌人击杀 |
| 瞬秒 | 被瞬间击杀 |
| 一穿三 | 一人击杀敌方全队 |

## 三、一局游戏的标准流程

```
大厅 → 配装 → 匹配队友 → 加载 → 进图 → 
搜物资/接敌 → 撤离/死亡 → 结算
```

| 阶段 | 时长 | 画面特征 | 内容价值 |
|------|------|---------|---------|
| 大厅 | 30s-2min | 角色展示、队伍界面 | 低（通常剪掉） |
| 配装 | 15-60s | 装备界面、武器选择 | 中（展示"破产"状态） |
| 进图 | 30s-3min | 第一人称跑图、搜刮 | 中（建立场景） |
| 接敌 | 15s-5min | 对枪、战术移动 | 🔥高（核心内容） |
| 撤离 | 10-30s | 倒计时、紧张等待 | 🔥高（悬念顶点） |
| 结算 | 5-10s | 收益/亏损展示 | 🔥高（情绪落点） |

## 四、抖音内容生态与风格

### 主流内容类型

| 类型 | 代表 | 特点 |
|------|------|------|
| **搞笑队友** | 破产兄弟 | 语音吐槽+失败合集，关键是队友之间的真实对话 |
| **逆袭翻盘** | 各式主播 | 0币开局→1000W撤离，爽文叙事 |
| **操作教学** | 辛巴达等 | 讲解枪械、地图点位、战术 |
| **整活/抽象** | 林树等 | 夸张剪辑+音效+表情包 |
| **剧情短剧** | 各种作者 | 用游戏画面演段子 |

### 破产兄弟风格拆解

参考视频「破产兄弟拯救水友」的核心要素：

1. **真人语音是灵魂** — 队友之间的真实对话："他好像看见咱了老大"、"我焯我专业夺舍的"
   - ❌ 不要用 AI 编对话替代真人语音
   - ✅ 保留全部原声，字幕只是辅助标注

2. **失败的戏剧性** — 视频好看不是因为操作好，而是因为死得搞笑
   - 受击特效（玻璃碎裂+「啊~」字）强化失败瞬间
   - 倒地/被偷袭/翻车都是好素材

3. **金额叙事线** — 顶部 "500W→1500W" 标题条贯穿全片
   - 给观众一个"赌注感"：这局能回本吗？
   - 结算时揭晓答案，形成叙事闭环

4. **长镜头保留语境** — 不像击杀集锦那样 0.5s 一切
   - 让观众看懂：谁在打谁、怎么死的、为什么好笑
   - 中位 3.1s/刀

5. **枪械后期特效** — 关键击杀/对枪时枪口发光
   - 不是游戏自带，是后期加的 VFX
   - 目的：强化击杀的"爽感"

## 五、剪辑指南

### 素材选择优先级

| 优先级 | 素材类型 | 原因 |
|--------|---------|------|
| 🔥🔥🔥 | 队友语音吐槽+搞笑死亡 | 最有传播力 |
| 🔥🔥 | 精彩击杀/一穿多 | 爽感内容 |
| 🔥🔥 | 撤离成功+高收益 | 情绪释放 |
| 🔥 | 撤离失败/破产 | 悬念/共情 |
| 🔥 | 配装展示（极穷/极富） | 建立叙事起点 |
| ⬜ | 跑图/搜物资 | 过渡填充 |

### 剪辑节奏

```
配装（建立目标）→ 进图（慢）→ 第一波接敌（中速）
→ 搞笑死亡/对话（高潮区，多切点）→ 撤离悬念（慢，保留紧张感）
→ 结算（情绪落点）
```

### 字幕风格

- 颜色：黄色字体+黑色描边（抖音通用风格）
- 位置：底部居中为主，重要吐槽句中置大字
- 内容：标注队友说了什么（不是编对话，是帮观众听清）
- 不要：emoji 表情、颜文字、过度煽情

### 顶部标题条

- 格式：「队名/主题」+ 金额目标
- 例：「破产兄弟+水友自救ing」500W→1500W
- 全片恒定显示，结算时揭晓结果

## 六、后期特效

### 枪械发光 (Weapon Glow)

- 在精彩击杀/关键对枪时间点叠加
- 特效层：枪口/枪身发光光晕（暖色#FF8800 或冷色#4488FF）
- 实现方式：ffmpeg overlay + 透明度动画
- 持续时间：0.5-1.0s

### 受击反馈

- 被击中时：屏幕边缘红色闪烁 + 玻璃碎裂素材
- 被击杀/倒地：画面抖动 + 大字特效
- 这些效果应该**配合游戏内实际事件触发**，不是随机加

## 七、常见踩坑

| ❌ 错误做法 | ✅ 正确做法 |
|---|---|
| AI 编对话配字幕 | 保留原声，字幕只标注 |
| 裁剪掉游戏 UI | 保留 UI，观众需要看血量/弹药/地图 |
| 0.5s 快切 | 保持 2-4s 让观众看懂 |
| 随便加 emoji | 用简洁文字，不卖萌 |
| 忽略游戏术语 | 用对术语（撤离不说"跑路"） |
| 配字不玩梗 | 了解"夺舍""得吃""白给"等社区梗 |
